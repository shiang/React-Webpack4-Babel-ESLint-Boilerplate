## React-Webpack4-Babel-ESLint-Boilerplate

### Features

- React 16.4.2
- Webpack 4
- ESLint with AirBnb Style Guide
- Hot module replacement
- Stage 2 ES Next features

### Installation

- git clone
- cd to the project
- npm install
- npm start
- Visit http://localhost:8080
